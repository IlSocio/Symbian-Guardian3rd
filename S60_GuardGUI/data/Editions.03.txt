Geben Sie einen Registrierungscode ein, um den erweiterten Funktionsumfang der Gold, Platinum oder Iridium Edition zu erhalten.

Wenn Sie die Gold Edition kaufen, können Sie:
+ Ihre Nachrichten, Ihr Telebuch und Ihren Kalender per Kennwort sperren.
+ jede installierte Anwendung per Kennwort sperren.
+ Guardian vor unbefugter Deinstallation schützen.
+ Informationen über die eingebuchte Telefontelle (cell-id), das Gebiet und dem Mobilfunkanbieter des Diebes erhalten.
+ unbegrenzt viele SIM-Karten der Liste der autorisierten SIM-Karten hinzufügen.
+ Technische Unterstützung zur Software erhalten.

Wenn Sie die Platinum Edition kaufen, können Sie:
+ alle Funktionen der Gold Edition verwenden.
+ Ihr Mobiltelefon mit Hilfe von GPS-Daten lokalisieren (benötigt GPS-Empfänger).
+ Verriegeln Sie das Telefon.
+ alle SMS-Nachrichten vom verlorenen Telefon wiedergewinnen.
+ Ihren kompletten Anrufverlauf wiedergewinnen.
+ alle Kontakte aus dem Telefonbuch wiedererhalten.
+ alle persönlichen Daten vom verlorenen Mobiltelefon löschen, wie zum Beispiel SMS,Kontakte, Bilder und Videos.
+ Und noch viel mehr... 

Wenn Sie die Iridinum Edition kaufen, können Sie:
+ alle Funktionen der Gold und Platinum Edition verwenden.
+ Erhalten in Echtzeit eine Kopie für den eingehenden und ausgehenden SMS.
+ Erhalten in Echtzeit den Protokollen für die ankommenden und abgehenden Anrufe.


Um eine ausführliche Beschreibung der Funktionen zu erhalten, besuchen Sie die offizielle Website.
Sie können die Software auf der offiziellen Website, oder direkt mit Ihrem Telefon kaufen: 
1) Wählen Sie Optionen > Kaufen und danach wählen die gwünschte Edition aus. 
2) Geben Sie Ihre Kreditkarteninformationen ein und bestätigen Sie die Zahlung.
3) Nach erfolgreicher Bezahlung können Sie Ihren Registrierungscode sehen. Diesen Code erhalten Sie auch per E-Mail.
4) Danach wählen Sie Optionen > Registrieren > Registrierungscode und geben Ihren Code ein.

Sobald Ihr Registrierungscode bestätigt wurde, wird Guardian zur gekauften Edition umgewandelt.
