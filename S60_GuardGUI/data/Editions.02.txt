Insérez le code d'enregistrement pour passer à la version Gold, Platinum ou Iridium.

En achetant Guardian Gold, vous pourrez:
+ Protéger par un mot de passe: SMS /  Annuaire téléphonique / Agenda.
+ Protéger par un mot de passe tous vos logiciels installés.
+ Protéger Guardian contre une désinstallation non autorisée.
+ Recevoir les renseignements suivants: Numéro du cell. / Région / Pays.
+ Ajouter un nombre illimité de SIM à la Liste des SIM autorisés.
+ Obtenir du soutien technique sur le logiciel.

En achetant Guardian Platinum, vous pourrez:
+ Utiliser toutes les fonctions de la version Gold.
+ Repérer avec précision le téléphone perdu en obtenant la latitude et la longitude du GPS.
+ Verrouiller le téléphone
+ Récupérer tous les SMS envoyés ou reçus du téléphone perdu.
+ Récupérer tous les journaux des appels faits ou reçus à partir du téléphone perdu.
+ Récupérer les contacts de l'annuaire téléphonique du téléphone perdu.
+ Effacer du téléphone perdu la totalité des SMS, des contacts, des photos et des vidéos.
+ Accéder à beaucoup d'autres fonctions...

En achetant Guardian Iridium, vous pourrez:
+ Utiliser toutes les Gold et les Platinum fonctions.
+ Recevez en temps réel une copie pour les sms entrants et sortants.
+ Recevez en temps réel les journaux pour les appels entrants et sortants.

Notre site Web officiel publie une description détaillée de l'ensemble des options.

Vous pouvez acheter le logiciel sur le site Web ou directement par téléphone:
1) Sélectionnez Options > Acheter et choisissez la version de votre choix.
2) Insérez les renseignements de votre carte de crédit et confirmer le paiement.
3) Une fois le paiement effectué, vous pourrez lire immédiatement votre code d'enregistrement, qui vous sera également envoyé par courriel.
4) Insérez votre code d'enregistrement en sélectionnant Options > S'enregistrer > Utiliser le code.

Dès que vous aurez entré votre code d'enregistrement, la version gratuite sera automatiquement convertie en la version achetée.
